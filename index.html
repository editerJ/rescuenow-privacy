한국어

1. 수집하는 정보

RescueNow는 다음 정보를 수집합니다:

  • 위치 정보: 긴급 구조 요청 시 현재 위치 (GPS)
  • 건강 정보: 사용자가 직접 입력한 혈액형, 알레르기, 복용 약물
  • 센서 데이터: 충격 감지를 위한 가속도계 데이터 (기기에만 저장)

2. 정보 사용 목적

  • 긴급 구조 요청 발송
  • 구조대원에게 의료 정보 제공
  • 자동 위험 감지 (충격/낙하)

3. 정보 공유

긴급 구조 요청 시에만 응급 서비스에 위치 및 건강 정보가 공유됩니다. 제3자 광고 목적으로 정보를 공유하지 않습니다.

4. 데이터 보관

건강 정보는 기기에만 저장되며, 앱 삭제 시 함께 삭제됩니다.

English

1. Information We Collect

  • Location: GPS location when emergency SOS is triggered
  • Health Info: Blood type, allergies, medications (entered by user)
  • Sensor Data: Accelerometer data for shock detection (stored on device only)

2. How We Use Information

  • Sending emergency rescue requests
  • Providing medical info to first responders
  • Automatic danger detection (shock/fall)

3. Information Sharing

Location and health information is only shared with emergency services when an SOS is triggered. We do not share data for advertising purposes.

4. Data Retention

Health information is stored on-device only and deleted when the app is uninstalled.
